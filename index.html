Projects

Learning Management System

A Python-based desktop application to manage and automate study and tasks. Desktop Learning Management System built with PyQt6 and SQLite — a compact, production-minded desktop app with role-based access (Student/Admin), full CRUD for courses and materials (PDFs with auto page-count + videos), and a card-style course dashboard with per-user progress indicators. It includes a custom PDF reader that saves the last-read page, student tasks and reminders, admin announcements with approval workflows, rich-text notes supporting inline images, and a color-coded event calendar. Built-in analytics (matplotlib) and auto-generated PDF reports (reportlab) make tracking and reporting simple

Hospital Management System

This Java-based desktop application, powered by MySQL and Swing, delivers a clean and intuitive interface for end-to-end hospital management. It packs in robust modules for patient, doctor, and staff administration, along with advanced room allocation and ambulance tracking systems. The built-in appointment scheduler ensures seamless booking and minimal conflicts. With real-time data handling, optimized workflows, and high-accuracy record management, it streamlines operations, saves valuable time, and keeps everything running like clockwork.
